# Java E-Commerce Console Application

A simple Java console-based e-commerce system built using Core Java (OOP, Collections, File Handling).  
Features include product handling, shopping cart operations, quantity updates, and price summaries.

## Features
- Add products with quantity  
- Calculate total cart price  
- Display clean cart summary  
- Modular code with OOP concepts  

## Tech Used
- Java (JDK 8+)
- OOP principles
- Collections (LinkedHashMap)
- Eclipse IDE

## Project Structure
src/
 ├── com.ecommerce.app  
 │      └── Main.java  
 ├── com.ecommerce.products  
 │      └── Product.java  
 └── com.ecommerce.services  
        └── CartService.java  

## How to Run
1. Clone the repo  
2. Open in Eclipse  
3. Run Main.java
